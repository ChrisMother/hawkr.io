technology

An industry first API that allows for direct access to any remote vending machine, with specific endpoints to:
Control mechanical functions of the machine, such as remote vending items, controlling vending speed of individual channels
Administer the machine, to update what stock is inside the machine
To retrieve data from the machine, such as order details, stock levels, and the current temperature
The API we’ve developed also supports aggregation across all machines to increase efficiencies:
Multicast special offers to all machines
Aggregate reporting
Update stock images
The real power in the API is its support for incorporating micro services. This means that ANY vending machine hardware and ANY payment hardware can be plugged in, without the need to change the core API. Therefore it is possible to use the Hawkr API with any vending machine setup.
There is no limit to what you can plugin to the API, for example, you could:
Hook vending machine activity into Twitter
Auto alert via email/sms/chat when stock is low or connection is lost
Add additional hardware for custom setups
